# Chaithanya Gudiwada Personal Website

Static personal portfolio website for Chaithanya Gudiwada, Senior DevOps Engineer.

## Local preview

Open `index.html` in a browser.

## GitHub Pages

To publish at the root user site URL, this folder should be pushed from a repository named:

`chaigudi.github.io`

When pushed to that repository's `main` branch, GitHub Pages will publish it at:

`https://chaigudi.github.io/`

If this code stays in `chaigudi/personal-website`, GitHub Pages will publish it as a project site:

`https://chaigudi.github.io/personal-website/`
